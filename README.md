react-boxplot
=============

Simple SVG box plots in React.


Features
------------

- Oh, it's really simple.


Installation
------------

    npm install react-boxplot


Development
-----------

1. Clone the repository.
2. `npm install`
3. `npm run watch`
4. `npm run open`
